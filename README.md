# SkillSquare - Find Skilled Professionals Near You 🛠️

SkillSquare is a comprehensive service platform designed to bridge the gap between people facing daily technical or household challenges and skilled professionals ready to solve them. 

Whether you need a reliable technician or want to register as a professional to offer your services, SkillSquare delivers a secure, smooth, and highly intuitive user experience.

## ✨ Key Features
- **User / Customer Portal:** Users can easily search, discover, contact, and book the top-rated technicians and professionals in their area to resolve daily problems.
- **Provider Portal:** Skilled workers can register, set up their profiles, and offer their specialized services to a broad audience.
- **Admin Dashboard:** A dedicated, secure administrative panel to manage the site, oversee users, and ensure platform quality.
- **Optimized UX:** The interface is built with a strong emphasis on user experience, ensuring that finding a professional or offering a service is as seamless as possible.

## 💻 Tech Stack & Architecture

Although this repository contains the **Frontend** codebase, SkillSquare is a complete, full-stack platform built with modern enterprise-level technologies.

### Frontend
- **Framework:** React.js powered by Vite for lightning-fast builds
- **Languages & Styling:** JavaScript, HTML5, CSS, Bootstrap
- **Mailing:** SMTP integration via EmailJS

### Backend (API)
- **Framework:** ASP.NET Core Web API (C#, .NET 9 & 10)
- **Architecture:** Multi-tier Architecture prioritizing clean design principles and scalability
- **Authentication & Security:** JWT (JSON Web Tokens) combined with ASP.NET Identity for robust role-based access control (Admin, User, Provider)

### Database
- **System:** SQL Server
- **Language:** SQL

### DevOps & Deployment
- **Frontend Hosting:** Vercel
- **Backend Hosting:** Microsoft Azure
- **Version Control:** GitHub

### Tools & Testing
- **API Documentation & Testing:** Swagger, Postman
- **Design Practices:** Adherence to standard UI/UX design principles and robust software design patterns.
